# Sistemas de Recomendação - Trabalho Final

Este repositório contém o desenvolvimento do trabalho final para a disciplina de **Sistemas de Recomendação** do curso de Ciência da Computação da **Universidade Federal Rural do Rio de Janeiro (UFRRJ)**.

O objetivo do projeto é implementar, hiperparametrizar e comparar duas abordagens de filtragem colaborativa (baseada em memória e baseada em modelo) utilizando a base de dados **MovieLens 100k**.
